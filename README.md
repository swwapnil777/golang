### Prerequisites

- Go installed on your machine

### Installation

1. Clone the repository:
    ```bash
    git clone <repository-url>
    ```

2. Navigate to the project directory:
    ```bash
    cd Go
    ```

### Running the Program

To run the Go program, use the following command:

```bash
go run main.go
